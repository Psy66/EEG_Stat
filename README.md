**EDF File Manager** is a user-friendly application with a graphical interface for managing files in the EDF (European Data Format). The program provides powerful tools for working with EDF files, including renaming, checking for corruption, finding duplicates, generating statistics, and more.

---

## ‚ú® Features

- **üìÇ Open Folder with EDF Files**: Select a directory to work with files.
- **üñãÔ∏è Rename EDF Files**: Automatically rename files based on metadata.
- **üö´ Remove Corrupted Files**: Find and delete corrupted EDF files.
- **üîç Remove Duplicates**: Find and delete duplicate EDF files.
- **‚è±Ô∏è Find Files with Similar Start Time**: Locate EDF files with similar recording start times.
- **üìä Generate Statistics**: Collect and visualize statistics for EDF files.
- **üìã Create Patient Table**: Generate a CSV table with patient names.
- **üé≤ Randomize Filenames**: Randomize filenames in the folder.
- **üë§ Remove Patient Info**: Remove patient information from EDF files.
- **üìÑ Read EDF File Info**: Display information about the selected EDF file.
